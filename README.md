Bkper Node.js client provides a simple and secure way to access the [Bkper REST API] on [Node.js](https://nodejs.dev/)

## Instalation

### Add the package:

```
npm i -S bkper-js
```
or
```
yarn add bkper-js
```
or
```
bun add bkper-js
```
